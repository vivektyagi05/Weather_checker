🌤️ Weather Checker – Real-Time Weather Web App
A modern and responsive weather web application built using HTML, CSS, and JavaScript, powered by the WeatherAPI to deliver real-time weather updates for any location worldwide.
🚀 Live Demo
👉 https://weather-checker-six-teal.vercel.app/⁠�
✨ Key Features
🌍 Real-Time Weather Data
Fetches live weather data using WeatherAPI
Supports global city search
Accurate temperature, humidity, wind, and condition
🔍 Smart Search Experience
Handles flexible input (e.g., Delhi, New York, Paris)
Input validation & error handling
Encoded queries for better API accuracy
🎨 Modern UI/UX
Clean and minimal glassmorphism design
Fully responsive (mobile + desktop)
Smooth user interaction
⚡ Performance & Reliability
Fast API response handling
Loading indicators for better UX
Graceful error messages

⚙️ How It Works
User enters a city name
App sends request to WeatherAPI
API returns real-time weather data
UI updates dynamically with:
Temperature 🌡️
Weather condition ☁️
Humidity 💧
Wind speed 🌬️
🔐 API Configuration
This project uses WeatherAPI:
Bash
https://api.weatherapi.com/v1/current.json
